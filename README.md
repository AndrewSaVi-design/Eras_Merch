# ERAS MERCH - Tienda Online

Bienvenido a la plataforma oficial de **Eras Merch**, una tienda de ropa personalizada diseñada para fans. Este proyecto utiliza tecnología moderna para ofrecer una experiencia de compra rápida y fluida.

## 🚀 Tecnologías utilizadas
* **Next.js 15+**: Framework de React para el frontend.
* **Tailwind CSS**: Para un diseño moderno y minimalista.
* **Lucide React**: Set de iconos elegantes.
* **Google Sheets API**: Usamos un Excel como base de datos dinámica.
* **PapaParse**: Para procesar los datos del inventario en tiempo real.
* **Vercel**: Hosting y despliegue continuo.

## 🛠️ Características principales
- **SSG (Static Site Generation)**: Carga instantánea de productos.
- **ISR (Incremental Static Regeneration)**: La web se actualiza sola cada 60 segundos si cambias algo en el Excel.
- **Carrito de compras**: Gestión local de pedidos.
- **Integración con WhatsApp**: Finalización de compra directa con el vendedor.
- **Diseño Responsive**: Optimizado para móviles y escritorio.

## 📦 Cómo actualizar el inventario
No necesitas tocar el código para cambiar productos. Solo edita el google sheet vinculado:
1. Agrega el nombre del artista y su foto.
2. Sube las imágenes de los polos a **PostImages**.
3. Pega los links, nombres y precios en la hoja de productos.
4. ¡La web se actualizará automáticamente en un minuto!
